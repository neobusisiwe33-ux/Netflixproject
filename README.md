This project is a Netflix-inspired landing page built using HTML and CSS. 
It recreates the look and feel of Netflix's homepage while applying core web development fundamentals.
The page includes a hero section with the Netflix logo, navigation bar, and a full-screen background banner, a Get Started form with an email input and button, a video trailer element, and a plans comparison table. 
At the bottom sits a fully functional FAQ section built with semantic HTML using the native details and summary tags, which allows questions to expand and collapse on click without any JavaScript.
The project is styled entirely with CSS — using custom variables, flexbox layouts, the box model, Google Fonts, and media queries to ensure the page is fully responsive on both desktop and mobile screens. 
All code is organised with comments throughout and includes a favicon in the browser tab.
